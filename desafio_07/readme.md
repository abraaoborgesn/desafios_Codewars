Escreva uma função que receba um inteiro não negativo (segundos) como entrada e retorne a hora em um formato legível ( HH:MM:SS)

HH= horas, preenchido com 2 dígitos, intervalo: 00 - 99
MM= minutos, preenchido com 2 dígitos, intervalo: 00 - 59
SS= segundos, preenchido com 2 dígitos, intervalo: 00 - 59
O tempo máximo nunca excede 359999 ( 99:59:59)
